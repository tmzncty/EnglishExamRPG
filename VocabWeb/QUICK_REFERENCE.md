# 🚀 VocabWeb 快速参考

## ⚡ 一分钟上手

```bash
# 1. 启动服务器
python server.py

# 2. 访问学习界面
电脑: http://localhost:8080
手机: http://你的IP:8080
```

## 🔧 常用命令

### 启动服务器
```bash
python server.py          # 命令行
start.bat                 # Windows 快捷方式
```

### 查看状态
```
http://localhost:8080/test.html
```

### 清除缓存
```
http://localhost:8080/clear.html
```

### 备份数据
```bash
copy user_vocab.db backup/vocab_20251213.db
```

---

## 🆘 常见问题快速解决

| 问题 | 解决方案 |
|------|---------|
| 一直加载 | 访问 `clear.html` 清除缓存 |
| 手机无法访问 | 检查同一 Wi-Fi，查看服务器 IP |
| 数据没同步 | 确认服务器运行，查看控制台 |
| 配置丢失 | 配置在本地，记录下来手动设置 |

## 💡 最佳实践

1. **首次使用**
   - 电脑先启动，配置好
   - 手机再连接，自动同步

2. **日常学习**
   - 碎片时间用手机
   - 深度学习用电脑
   - 睡前复习错题

3. **数据安全**
   - 定期备份 `user_vocab.db`
   - 重要数据导出
   - 清缓存前确认同步
