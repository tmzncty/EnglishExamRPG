# 背景图片规格说明 (Updated to v3.2)

## 文件夹结构
```
assets/backgrounds/
├── horizontal/   # 横版背景（PC/平板横屏）
└── vertical/     # 竖版背景（手机/平板竖屏）
```

## 推荐尺寸

### 横版背景 (horizontal/)
- **推荐尺寸**: 1920 x 1080 px (16:9)
- **最小尺寸**: 1280 x 720 px
- **格式**: JPG 或 PNG（推荐 JPG，文件更小）
- **用途**: PC 桌面、平板横屏模式

### 竖版背景 (vertical/)
- **推荐尺寸**: 1080 x 1920 px (9:16)
- **最小尺寸**: 720 x 1280 px
- **格式**: JPG 或 PNG
- **用途**: 手机、平板竖屏模式

## 命名规范
- 使用英文或数字命名，避免中文和特殊字符
- 示例: `bg_sakura_01.jpg`, `bg_library.png`, `bg_night_city.jpg`

## 风格建议
适合二次元风格的背景主题：
1. 🌸 樱花校园
2. 📚 图书馆/书房
3. 🌃 夜景城市
4. 🏫 教室
5. ☕ 咖啡厅
6. 🌅 黄昏天空
7. 🎮 游戏房间
8. 🌌 星空

## 注意事项
1. 背景不要太亮或太杂乱，以免影响阅读
2. 颜色偏暗或带蓝/紫色调更适合深色主题
3. 可以带轻微模糊效果，减少视觉干扰
4. 建议文件大小控制在 500KB 以内（压缩后）

## 放置完成后
把图片放进对应文件夹后，程序会自动读取并随机显示。
